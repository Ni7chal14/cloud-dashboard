# ☁️ Cloud Cost & Carbon Footprint Calculator

An interactive web-based dashboard that compares major cloud providers — AWS, GCP, and Azure — based on **cost, energy usage, and environmental impact (CO₂ emissions)**.

---

## 🚀 Live Demo

👉 [View Project](https://your-link-here)

---

## 📌 Features

* 💰 **Real-time Cost Estimation**

  * Compute, Storage, and Data Transfer pricing
* 🌱 **Carbon Footprint Calculation**

  * Estimates CO₂ emissions based on usage
* 📊 **Interactive Charts**

  * Cost breakdown visualization
  * Green score vs emissions comparison
* ⚡ **Dynamic Inputs**

  * Adjust vCPU, RAM, storage, and usage hours
* 🏆 **Smart Recommendation System**

  * Suggests the best cloud provider based on cost & sustainability

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Chart.js (for data visualization)

---

## 🧠 How It Works

* Cost is calculated using predefined pricing models for each cloud provider
* Energy consumption is estimated based on vCPU usage
* CO₂ emissions are derived using energy-to-carbon conversion factors
* A **green score system** evaluates sustainability of providers
